## 🔑 Key Points

* OSI = **7 layers** showing how data moves in a network
* Think: **Top (user) → Bottom (wire)**

### Layers (7 → 1)

* **7️⃣ Application** → 🌐 HTTP, DNS (user actions)
* **6️⃣ Presentation** → 🔐 Encryption (TLS/SSL)
* **5️⃣ Session** → 🔄 Start/stop connection
* **4️⃣ Transport** → 🚚 TCP/UDP, ports
* **3️⃣ Network** → 📍 IP address (routing)
* **2️⃣ Data Link** → 🧾 MAC address (local delivery)
* **1️⃣ Physical** → 🔌 Cables, signals, bits

## 🎯 Core Anchors (MOST IMPORTANT)

* **Layer 7 = WHAT** → 🌐 (HTTP, DNS)
* **Layer 4 = HOW** → 🚚 (TCP/UDP, ports)
* **Layer 3 = WHERE** → 📍 (IP address)

👉 If you remember ONLY these → you're solid

---

## ⚡ Mental Model (Easy Story)

* L7 → You **write message** ✍️
* L4 → Choose **delivery method** 🚚
* L3 → Add **address (IP)** 📍
* L2 → Local delivery (MAC) 🧾
* L1 → Send through wire 🔌

---

## 🛡️ Why It Matters for SOC

* **L7** → Web attacks (login abuse, HTTP)
* **L4** → Suspicious ports 🚨
* **L3** → Track attacker IP 📍
* Used in logs, alerts, investigations
